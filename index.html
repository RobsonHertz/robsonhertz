O que é HTML?

HTML significa HyperText Markup Language. É a linguagem padrão para criar páginas web.

Estrutura Básica de um Documento HTML

  • <!DOCTYPE html>: Declara o tipo de documento.
  • <html>: Elemento raiz que envolve todo o conteúdo.
  • <head>: Contém metadados como título e links de estilo.
  • <body>: Contém o conteúdo visível da página.

Exemplo de Código

<!DOCTYPE html>
<html>
<head>
    <title>Exemplo</title>
</head>
<body>
    <h1>Olá Mundo!</h1>
</body>
</html>
